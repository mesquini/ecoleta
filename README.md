## :information_source: How To Use

```bash
# Clone this repository
$ git clone https://github.com/mesquini/ecoleta

> cd ecoleta

> cd backend
> yarn install
> yarn dev
> yarn queue

> cd frontend
> yarn install
> yarn start
```
